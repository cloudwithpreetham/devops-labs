# 12-Configuration-Management (Ansible)

This repository contains my hands-on practice and mini-projects completed while learning **Ansible** as part of my **90 Days of DevOps** journey. Each day focuses on a different Configuration Management concept, progressing from Ansible basics to real-world infrastructure automation and project implementation.

## Learning Objectives

- Understand Infrastructure as Code (IaC) using Ansible
- Automate Linux server configuration
- Manage multiple servers with inventories
- Build reusable Ansible Playbooks and Roles
- Secure sensitive information using Ansible Vault
- Implement Configuration Management best practices
- Deploy real-world applications using Ansible
